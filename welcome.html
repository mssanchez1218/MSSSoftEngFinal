Welcome!

MicrochipBuilder is a website meant to show users what goes into building a computer and how to go about doing it and future upkeep.

The goal of this is to help users be more comftorable being able to do this sort of stuff and help them save money and have an overall better experience building something themselves, theres nothing better than seeing your PC you built yourself boot up for the first time.

To navigate this site you can continue using the arrows at the bottom to learn from the start, but if you are a returning user or even already know some of the information, you can use the menu at the top of each page.
