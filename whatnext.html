What Next???

Congrats!
You built your own PC but now what? first we should install an operating system. Most users will be using windows which i recommend using G2A for becuase it is much cheaper than retail versions of windows. all thats left now is installing any software or files from your old devices and installing programs such as Chrome, Firefox, anitvirus and such.
